Learning statistics

Coursera now has some really great (and free!) classes for learning statistics, including classes on Mathematical Biostatistics, Data Analysis, and Computing for Data Analysis.


Some other fun and useful resources include Matthias Vallentin's Probability and Statistics Cookbook, A Handbook of Statistical Analysis Using R by Everitt and Hothorn, and Networks, Crowds, and Markets by Easley and Kleinberg.
