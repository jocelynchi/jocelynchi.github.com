Purpose

This is a step-by-step tutorial to posting the Getting to the Bottom articles on Statisticsviews.com. We use the first article from the series as an example. The files used in this example can be found in gradientdescent-chi-v2.zip.


Step-by-Step Tutorial


Step 1

Open the html file containing the blog template for Statisticsviews.com in a text editor. Insert the following code inside the <head>...</head> section of the template. 

<!-- Knitr CSS -->
<link href="http://jocelynchi.com/index_files/knitr.css" rel="stylesheet">

Step 2

Copy and paste the following code directly before the </head>tag in the template html file. 

<!-- R syntax highlighter -->
<script type='text/javascript' src="http://jocelynchi.com/gettingtothebottom/r-syntax-highlighter.js"></script>

<!-- MathJax scripts -->
<script type="text/javascript" src="https://c328740.ssl.cf1.rackcdn.com/mathjax/2.0-latest/MathJax.js?config=TeX-AMS-MML_HTMLorMML">
</script>

Save and close the template file.

Step 3

Open the html editor for posting a new blog post to Statisticsviews.com.

Extract files from gradientdescent-chi-v2.zip and open web_graddescent.html in a text editor. Copy everything between the <body>...</body> tags in web_graddescent.html and paste into the html editor as a new post. (Note: If you are not editing the post inside the html editor, the html tags may not render correctly.)

Save and check that the post looks correct before publishing. All the material should render correctly but you may want to adjust the CSS styling to taste. On my site, the post looks like this.
